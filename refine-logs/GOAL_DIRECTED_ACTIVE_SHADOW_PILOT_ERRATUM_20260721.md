# Goal-Directed ActiveShadow Pilot 0 sample-availability erratum

The first label-blind freezer run stopped before writing a protocol because it
found 16 eligible public events, below the planned minimum of 20. The diagnostic
reported only public eligibility counts: 16 eligible and 354 without a frozen
pre-observation witness channel. No Rescue/Reverse label or branch outcome was
read, and no event was collected.

Pilot 0 now collects all 16 eligible events and uses a minimum of 12 total
events with at least 3 events per class. Rescue drop (at most 2%), Reverse recall
(at least 20%), the requirement for at least one query-incremental Reverse hit,
and the one-query cost ceiling are unchanged. The reduced pilot is explicitly
an applicability-conditioned feasibility check and cannot support a paper-facing
or formal risk claim.
