# RAPG Pilot 0 实验计划

## 目标与声明边界

本实验只验证：在固定 ToolSandbox A/B Full-Shadow outcome bank 上，RAPG 是否能用 20% expected Shadow budget 无偏重建冻结策略的 projected proposal gradient，并比 uniform auditing 降低至少 15% gradient MSE。

Pilot 0 不训练策略，不证明 unassisted task success，也不把旧 V4.4 的 DeepSeek proposal 冒充本地 LoRA policy sample。

## 受控行为策略

对每个 V4.4 public event，固定候选集合 `{action_a, action_b}`。冻结 LoRA checkpoint 对两个完整 canonical action sequence 计算 mean log-prob score，并定义

```text
pi_k(j|x,C) = softmax(score_j / temperature).
```

使用 hash-bound seed 从该候选策略真实采样 proposal index。Harness default 为 `action_b`：采到 A 时发生 replacement，采到 B 时直接观察 B return。由于 proposal 在 Pilot 运行时由这个冻结策略采样，旧 bank 的 DeepSeek A 生成身份不被错误复用。

## 数据

- public: `outputs/toolsandbox_v44_candidate_diversity_seed42/data/events.public.jsonl`
- private Full Shadow: `outputs/toolsandbox_v44_candidate_diversity_seed42/data/outcomes.private.jsonl`
- 预期：126 events，按 `task_id_hash` 做 deterministic cross-fitting。
- ground truth：官方 paired branch `terminal_similarity`，不是另一个模型输出。

## Milestones

| Milestone | 内容 | Gate | 预算 |
|---|---|---|---:|
| M0 | synthetic estimator sanity + unit tests | exact identity/unbiasedness tests pass | CPU < 5 min |
| M1 | LoRA candidate-policy score/sketch extraction | all events bound; no NaN; public/private hashes match | 1 GPU, 1–3 h |
| M2 | 1,000 fixed-propensity audit resamples | integrity checks all true | CPU < 10 min |
| M3 | Pilot gate | projected bias ≤5%; MSE gain ≥15%; beats residual-only | no training |

## Compared allocation rules

1. Uniform.
2. Residual-only: predicted `sigma`.
3. Score-only: exact LoRA score norm `w`.
4. RAPG: `w * predicted_sigma`.
5. Oracle: `w * abs(Y_A-m)`; unattainable ceiling only.

All rules use the same fixed `c=1`, 20% expected audit rate, and pre-registered `p_min=0.05`. Primary estimates use committed `1/p` with no clipping.

## Public feature and leakage rule

The augmentation/allocation model sees only hashed public prompt/action features, frozen candidate-policy scores/probabilities, proposal index, and the executed B return. It never sees A outcome on held-out tasks before propensities are fixed. Mean and residual-second-moment models are task-group cross-fitted.

## Metrics

- projected-gradient relative bias and 5% equivalence gate;
- projected-gradient MSE and gain over Uniform;
- cosine similarity to Full-Shadow sampled-batch gradient;
- expected/realized cost, ESS, maximum `1/p`;
- task-group improvement fraction and concentration;
- all artifact and model identities.

## Stop rules

- Any public/private/model hash mismatch: stop.
- Fewer than two task groups per fold or no replaced events: stop.
- Primary propensity clipping: stop.
- RAPG MSE gain below 15% or not better than residual-only: stop before policy training.
- Positive policy claim remains unauthorized until a later fresh-task training experiment passes.
